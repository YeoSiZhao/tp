# InventoryDock

InventoryDock is a high performance Command Line Interface (CLI) application designed for professional warehouse 
managers and inventory specialists who prefer the speed and efficiency of keyboard-driven workflows.

While traditional inventory management systems often rely on heavy graphical interfaces, InventoryDock focuses on 
rapid data entry and retrieval through a streamlined text-based interface, allowing users to manage stocks with 
minimal friction.
